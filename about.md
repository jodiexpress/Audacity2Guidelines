# <cy>Audacity2Guidelines</cy>

Convert your <cy>Audacity</cy> label tracks into <cg>Geometry Dash</cg> guidelines in just a few clicks — no manual tapping required.

## How to use

1. In <cy>Audacity</cy>, add labels to your track where you want guidelines
2. Export labels: <cy>File → Export → Export Labels</cy>
3. Open a level in the <cg>GD Editor</cg>
4. Go to the <cy>Guidelines Creator</cy>
5. Click <co>**"Import Audacity"**</co>
6. Select your <cy>**.txt label file**</cy>
7. Done!

## <cr>IMPORTANT!</cr>

For colored guidelines, name your labels in <cy>Audacity</cy> like this:

- Label named <co>**orange**</co> → Orange guideline
- Label named <cy>**yellow**</cy> → Yellow guideline  
- Label named <cg>**green**</cg> → Green guideline
- Any other name → Orange guideline (default)

## Notes

- Multiple imports will be <cy>merged</cy> with existing guidelines
- Use the <cr>**Clear**</cr> button to reset all guidelines
